# ORKES · Social Media Assets
## Estrutura da pasta

```
social-media/
├── logo/
│   └── logo-quadrada.jpg        ← Logo 1:1 para todas as plataformas
│
├── instagram/
│   ├── foto-perfil.jpg          ← Foto de perfil (logo quadrada)
│   ├── post-01-apresentacao.jpg ← Primeiro post de apresentação
│   └── textos.md                ← Bios, legendas e hashtags
│
├── facebook/
│   ├── foto-perfil.jpg          ← Foto de perfil da Page
│   ├── banner-capa.jpg          ← Imagem de capa (usar 820×312px)
│   └── textos.md                ← Descrição da página e posts
│
├── linkedin/
│   ├── foto-perfil.jpg          ← Logo da empresa (300×300px mín.)
│   ├── banner-capa.jpg          ← Banner da Company Page (1128×191px)
│   └── textos.md                ← Descrição, especialidades e posts
│
└── olx/
    └── anuncio.md               ← Título, descrição e instruções do anúncio
```

## Identidade Visual

| Token       | Valor      |
|-------------|------------|
| Fundo       | `#000000`  |
| Destaque    | `#00E5FF`  |
| Texto       | `#ffffff`  |
| Fonte base  | Inter / sans-serif moderna |

## Checklist de publicação

### Instagram
- [ ] Criar conta @orkes.ti (ou similar disponível)
- [ ] Configurar como conta profissional/empresa
- [ ] Subir foto-perfil.jpg
- [ ] Preencher bio com texto de textos.md
- [ ] Adicionar link do site na bio
- [ ] Publicar post-01-apresentacao.jpg com legenda de textos.md

### Facebook Page
- [ ] Criar Page em facebook.com/pages/create
- [ ] Categoria: Empresa de serviços de TI
- [ ] Subir foto-perfil.jpg e banner-capa.jpg
- [ ] Preencher "Sobre" com texto de textos.md
- [ ] Adicionar CNPJ, email, WhatsApp e site
- [ ] Publicar post de boas-vindas e fixar no topo

### LinkedIn Company Page
- [ ] Criar em linkedin.com/company/setup/new
- [ ] Setor: Tecnologia da Informação e Serviços
- [ ] Subir foto-perfil.jpg e banner-capa.jpg
- [ ] Preencher "Sobre" com texto de textos.md
- [ ] Adicionar especialidades
- [ ] Publicar post de apresentação

### OLX
- [ ] Acessar olx.com.br > Anunciar > Serviços > TI
- [ ] Usar título e descrição de anuncio.md
- [ ] Definir preço como "Sob consulta" ou "A partir de R$ 500"
- [ ] Adicionar fotos: logo + screenshot do site + foto profissional
- [ ] Ativar opção de contato por WhatsApp
